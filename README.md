# Projeto de Estudo - Qwen com Vercel

Este repositório contém projetos desenvolvidos para estudo e experimentação com a IA **Qwen**, utilizando a plataforma **Vercel** para deploy e hospedagem.

## 📁 Estrutura do Projeto

- `index.html` - Página principal
- `flappy-bird.html` - Jogo Flappy Bird
- `ozzy-mario.html` - Jogo Ozzy Mario (padronizado)
- `consorcio-landing/` - Landing page de consórcio
- `ozzy-game/` - Versão do jogo Ozzy
- `ozzy-game-clean/` - Versão limpa do jogo Ozzy

## 🛠️ Principais Linguagens e Tecnologias Utilizadas

- **HTML5** - Estruturação das páginas e jogos
- **CSS3** - Estilização e animações
- **JavaScript** - Lógica dos jogos e interatividade
- **Canvas API** - Renderização dos jogos no navegador
- **Vite** - Build tool para otimização (em alguns projetos)
- **Vercel** - Plataforma de deploy e hospedagem

## 🎯 Objetivo

Este projeto tem como finalidade estudar e experimentar as capacidades da IA Qwen na criação de aplicações web, jogos e landing pages, utilizando tecnologias modernas de desenvolvimento front-end.

## 🚀 Deploy

Todos os projetos estão configurados para serem implantados na **Vercel**, permitindo acesso rápido e fácil às aplicações desenvolvidas.
