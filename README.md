## 📖 Sobre o Projeto

Repositório com **jogos interativos e projetos web** desenvolvidos com o auxílio do **Claude (Anthropic)** como co-piloto de desenvolvimento, utilizando o **VS Code** como editor principal.

Os projetos são hospedados na **Vercel** para acesso rápido e deploy contínuo.

## 🚀 Como Rodar Localmente

```bash

git clone https://github.com/jecdurvalo/jecdurvalo.git

# Acesse a pasta
cd jecdurvalo

# Abra o index.html no navegador ou use um servidor local
npx serve .
```
